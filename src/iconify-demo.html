Usage in TypeScript

You can also create icons programmatically in your TypeScript components:

// Create an icon element const icon = document.createElement('iconify-icon'); icon.setAttribute('icon', 'mdi:check-circle'); icon.style.color = 'green'; icon.style.fontSize = '24px'; // Add to container container.appendChild(icon);
