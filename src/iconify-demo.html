Button Examples

Icons work great in buttons and interactive elements:

import PlusIcon from '~icons/mdi/plus?raw'; const button = document.createElement('button'); button.className = 'demo-button'; button.innerHTML = `${PlusIcon} Add Item`;

Size Examples

Control icon size with CSS width/height:

import HeartIcon from '~icons/mdi/heart?raw'; const icon = document.createElement('span'); icon.innerHTML = HeartIcon; const svg = icon.querySelector('svg'); if (svg) { svg.style.width = '32px'; svg.style.height = '32px'; }

Color Examples

Style icons with CSS color property (uses currentColor):

import StarIcon from '~icons/mdi/star?raw'; const icon = document.createElement('span'); icon.innerHTML = StarIcon; const svg = icon.querySelector('svg'); if (svg) { svg.style.color = '#f39c12'; }

Usage in TypeScript

Import icons as ES modules at build time:

// Import icons import CheckCircleIcon from '~icons/mdi/check-circle?raw'; import SettingsIcon from '~icons/mdi/settings?raw'; // Create helper function function createIcon(iconSvg: string, size = '24px'): HTMLElement { const wrapper = document.createElement('span'); wrapper.className = 'icon-wrapper'; wrapper.innerHTML = iconSvg; const svg = wrapper.querySelector('svg'); if (svg) { svg.style.width = size; svg.style.height = size; } return wrapper; } // Use it const icon = createIcon(CheckCircleIcon, '32px'); container.appendChild(icon);
